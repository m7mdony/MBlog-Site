### **كيفية الاستخدام في المشروع:**

#### **1. استيراد المكون:**
```tsx
import ContentCard from './components/ContentCard';
```

#### **2. في صفحات المفاهيم:**
```tsx
<ContentCard
  title={concept.title}
  imageUrl={concept.imageUrl}
  views={concept.views}
  likes={concept.likes}
  comments={concept.comments}
  onClick={() => navigateToConcept(concept.id)}
/>
```

#### **3. في صفحات الإنفوجرافيك:**
```tsx
<ContentCard
  title={infographic.title}
  imageUrl={infographic.thumbnailUrl}
  views={infographic.views}
  likes={infographic.likes}
  comments={infographic.comments}
  isInfographic={true}
  onClick={() => navigateToInfographic(infographic.id)}
/>
```

#### **4. في لوحة التحكم (قابلة للسحب):**
```tsx
<ContentCard
  id={item.id}
  index={index}
  title={item.title}
  imageUrl={item.imageUrl}
  views={item.views}
  likes={item.likes}
  comments={item.comments}
  isDraggable={true}
  showDragHandle={true}
  onOptionsClick={() => handleOptions(item.id)}
/>
```

## 📚 الملفات الجديدة المضافة

### **ContentCard.tsx (محدث ومحسن)**
- مكون بطاقة محتوى متقدم مع دعم كامل للإنفوجرافيك والسحب والإفلات
- تصميم عصري مع Tailwind CSS وتأثيرات بصرية متقدمة
- دعم كامل للوصول (accessibility) والتفاعل
- قابل للنقر بالكامل مع callbacks قابلة للتخصيص
- **الميزات الجديدة:**
  - `isDraggable`: تفعيل/تعطيل السحب والإفلات
  - `showDragHandle`: إظهار/إخفاء مقبض السحب
  - `id` و `index`: مطلوبان للسحب والإفلات
  - تأثيرات بصرية متقدمة أثناء السحب
  - منع التداخل بين النقر والسحب

### **ContentCardExample.tsx (محدث)**
- ملف أمثلة شامل لاستخدام مكون ContentCard
- أمثلة متنوعة للبطاقات العادية والإنفوجرافيك
- أمثلة للبطاقات القابلة للسحب والإفلات
- أمثلة للبطاقات بدون مقبض سحب مرئي
- توضيح جميع الخصائص والاستخدامات الممكنة

### **أمثلة الاستخدام:**

#### **بطاقة عادية:**
```tsx
<ContentCard
  title="عنوان المقال"
  imageUrl="/path/to/image.jpg"
  views={1000}
  likes={50}
  comments={10}
  onClick={() => console.log('تم النقر')}
/>
```

#### **بطاقة إنفوجرافيك:**
```tsx
<ContentCard
  title="عنوان الإنفوجرافيك"
  imageUrl="/path/to/image.jpg"
  views={2000}
  likes={100}
  comments={25}
  isInfographic={true}
  onClick={() => console.log('تم النقر')}
/>
```

#### **بطاقة قابلة للسحب:**
```tsx
<ContentCard
  id="unique-id"
  index={0}
  title="عنوان قابل للسحب"
  imageUrl="/path/to/image.jpg"
  views={1500}
  likes={75}
  comments={15}
  isDraggable={true}
  onClick={() => console.log('تم النقر')}
/>
```

#### **بطاقة قابلة للسحب بدون مقبض:**
```tsx
<ContentCard
  id="unique-id"
  index={1}
  title="عنوان قابل للسحب بدون مقبض"
  imageUrl="/path/to/image.jpg"
  views={1200}
  likes={60}
  comments={12}
  isDraggable={true}
  showDragHandle={false}
  onClick={() => console.log('تم النقر')}
/>
```
